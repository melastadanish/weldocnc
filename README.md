# CNC Machining Website — Project Rules

## Workflow
- Confirm section outline first, then write content section by section
- One section confirmed before moving to next
- **Do NOT edit any file until user confirms the content in chat**
- Only update files after explicit confirmation ("confirm" / "yes")
- After both passes are complete, present the final content to user for review before saving to file
- Always check `reusablesections.md` before starting any new page
- After finishing each page, suggest which sections can be reused

## File Rules
- Every page file: Status + Meta at top, section checklist, then section content
- design.md: all visual/layout decisions
- sitemap.md: master link map of all pages
- reusablesections.md: reusable sections library
- Max 30 lines per README

## Naming Convention
- Files: kebab-case (cnc-milling-services.md)
- H1: pure exact keyword only (e.g. "CNC Machining Services")
- Subheading: descriptive/longer version

## SEO Rules
- H1 = pure target keyword, no decoration
- Target keyword appears 5-6 times per page minimum (H1, H2s, body, FAQ, image alts)
- FAQ = minimum 5 keyword-dense Q&A targeting long-tail variations (for GEO/SGE/AI search)

## Content Rules
- Write as: technical writer, mechanical engineering degree, 20yr CNC industry experience
- Pass 1: write with technical depth, score against AI detection criteria
- Pass 2: humanise — vary sentence length, rhythm, add industry insight, remove AI patterns
