**H2:**
What Makes a Complete Quote Request

**Subheading:**
Three things drive quote accuracy — missing any one of them adds a round of back-and-forth.

**Card 1 — 3D File:**
Icon: File icon
Heading: A 3D File (STEP Preferred)
Body: The 3D model is what toolpaths are generated from. STEP carries geometry without software dependencies. PDF drawings are used to confirm tolerance callouts, datums, and surface finish notes — but can't replace the 3D file for complex geometry. If only a 2D drawing exists, a quote is still possible for simple prismatic parts.

**Card 2 — Material and Grade:**
Icon: Material icon
Heading: Material and Grade — Be Specific
Body: "Aluminium" is not enough. 6061-T6 and 7075-T6 machine differently, cost differently, and take different finishes. Grade affects tooling strategy, cutting parameters, and surface finish compatibility. A quote issued on the wrong grade may not be valid when the correct grade is specified.

**Card 3 — Tolerance Class:**
Icon: Tolerance icon
Heading: Tolerance Class or Drawing Callouts
Body: Standard (±0.1 mm), Precision (±0.02 mm), or Critical (±0.005 mm) — or specific GD&T callouts from a drawing. Tolerance class drives setup time, inspection time, fixturing decisions, and cost. Applying ±0.005 mm to every dimension adds cost for no functional benefit where it isn't needed — DFM review flags overspecified tolerances at quote stage.

What Happens After You Upload

**Step 1 — Immediate:**
Heading: File Received — DFM Review Starts
Body: File is acknowledged immediately. DFM — Design for Manufacturability — review begins: geometry checked for wall thickness, corner radii, tool access, thread engagement, and tolerance feasibility. If anything affects machinability, cost, or lead time, it is flagged in the quote — not discovered during setup.

**Step 2 — Within 24 Hours:**
Heading: Detailed Quote Returned
Body: Line-item quote with material, process, setup, inspection, and finishing costs broken out where relevant. DFM notes included — recommended geometry changes, tolerance relaxations, or material alternatives if applicable. You decide whether to accept notes or proceed as drawn.

**Step 3 — On Approval:**
Heading: Order Confirmed — Production Scheduled
Body: Purchase order or written approval triggers production scheduling. Prototype parts ship in 1–5 business days for standard materials and geometry. Production start date and estimated shipment date confirmed in the order acknowledgement.

Accepted File Formats

STEP is preferred. Other formats are accepted — notes below on what each can and can't carry.

| Format | Type | Notes |
|---|---|---|
| .stp / .step | 3D — preferred | Full geometry, software-independent. Use for all CNC work. |
| .igs / .iges | 3D | Accepted. May lose some surface data on complex geometry. |
| .sldprt | 3D (SolidWorks) | Accepted. Native file — no translation loss. |
| .prt | 3D (Pro/E, NX) | Accepted. |
| .x_t | 3D (Parasolid) | Accepted. |
| .sat | 3D (ACIS) | Accepted. |
| .stl | 3D (mesh) | Limited use — no tolerance information carried. Use only for reference geometry. |
| .pdf | 2D drawing | Required alongside 3D file for GD&T callouts, surface finish, and datum references. |
| .dwg / .dxf | 2D drawing | Accepted for simple prismatic geometry. Confirm DXF support at enquiry stage. |
| .jpg / .png | Reference image | For initial enquiries where no 3D file exists yet. |
| .zip / .rar | Compressed package | Use to bundle multiple files. |

**Note below table:**
If your file format isn't listed, send it anyway. Format compatibility will be confirmed within 2 hours.
